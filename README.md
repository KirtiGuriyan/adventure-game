# Adventure Game (Python)

A simple text-based adventure game built in Python where players make choices to explore a forest or a cave.

---

## Author
Kirti Guriyan

---

##  Description
This is a beginner-friendly **choice-driven RPG game** where the player:
- Starts at a crossroads
- Chooses between forest or cave
- Makes decisions that affect the story outcome
- Can replay the game multiple times

---

##  Features
- Interactive command-line gameplay
- Multiple story paths (forest & cave)
- Replay option
- Simple decision-based logic
- Beginner-level Python project

---

##  Game Flow
1. Enter player name
2. Choose path:
   - Forest 🌲
   - Cave 🕳️
3. Continue adventure based on choices
4. Reach different outcomes
5. Restart or exit

---

##  Technologies Used
- Python 3
- Basic input/output
- Conditional statements
- Loops and functions

---

##  Installation & Setup

To play **The Adventure Game**, ensure you have **Python 3.x** installed.

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/KirtiGuriyan/adventure-game.git
    ```
2.  **Navigate to the Project Folder:**
    ```bash
    cd adventure-game
    ```
3.  **Run the Game:**
    ```bash
    python adventure_game.py
    ```
